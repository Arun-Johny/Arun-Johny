# 🐍 Contribution Snake

> Create `.github/workflows/snake.yml`

After enabling GitHub Actions, add:

```yaml
name: Generate Snake

on:
  schedule:
    - cron: "0 */12 * * *"

jobs:
  build:
    runs-on: ubuntu-latest

    steps:
      - uses: Platane/snk/svg-only@v3
        with:
          github_user_name: Arun-Johny
          outputs: dist/github-contribution-grid-snake.svg
```

Then display:

```html

</p>
```
